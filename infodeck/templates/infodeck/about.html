About InfoDeck

InfoDeck es una aplicacion web para poder conocer los ultimos cambio que tienen tus diapositivas de slideWiki. Permitiendo al usuario realizar consultas de sus decks de forma simple y organizada, para así reducir el tiempo que este perdería en buscar slide por slide posibles cambios en las slides de donde se ha reutilizado.

Usos

Los usos de esta aplicacion se pueden ver en el buscador.

http://infodeck.herokuapp.com/

La cual tiene la siguiente apariencia:


La busqueda se tiene que realizar por id, la cual podemos encontrar en la url de nuestro deck.

https://slidewiki.org/deck/{id}/...

Y el resultado sera una lista con información del contenido del deck.


Esta aplicacion ademas hace uso de una API que hemos creado desde los datos obtenido por la API de slideWiki.Esta API que hemos creado consta de una sola operación de tipo GET, la cual nos dara mas informacion acerca del contenido del deck. Para poder acceder a esta solo hace falta hacer una llamada con la id del deck.

http://infodeck.herokuapp.com/api/deck/{id}
